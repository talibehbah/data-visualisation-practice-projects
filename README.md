# Data Visualisation Practice Projects

This repository contains practice projects for **R Shiny** and **Power BI**, with Shiny apps ready to run in RStudio without any setup.
## Projects

### R Shiny
- **Australian Energy Dashboard:** Open `rshiny/rshiny.Rproj` in RStudio, then click **Run App**. You may need to install the required packages for the first run.

### Power BI
- **AdventureWorks Practice Reports:** Various reports based on AdventureWorks datasets.
- **Maven Market Reports:** Practice reports analyzing Maven Market data.

## How to Run

### R Shiny
1. Open `rshiny/rshiny.Rproj` in RStudio.
2. Click **Run App**. The app is fully functional without additional configuration.

### Power BI
1. Open any `.pbix` file in Power BI Desktop.
2. Some of the raw datafiles are available in the dataset folder.
